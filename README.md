"# Analise_Ecommerce" 
 Análise Exploratória de Dados — Ecommerce Estatística
Este projeto realiza uma Análise Exploratória de Dados (EDA) utilizando um dataset de produtos de ecommerce, contendo informações como:
- Preço
- Nota média
- Número de avaliações
- Desconto
- Marca
- Material
- Gênero
- Temporada
- Quantidade vendida (codificada)
O objetivo é identificar padrões, relações e insights relevantes sobre o comportamento dos produtos.

🚀 Tecnologias Utilizadas
- Python 3
- Pandas
- Seaborn
- Matplotlib
